User story:

After a student logs in, they will given a list of all enrolled units, within each of those units, there will be a list of all tasks required to be completed for the unit.
The units will have the unit title, unit code, and the trimester/year enrolled.
Each unit will have a list of tasks.
Each task will have a task title, task grade and due date.

Requirements:
Unit class
Task class
Function to get all Units
Function to get Unit Title/Unit Code/Trimester Enrolled
Function to get all Tasks
Function to get Task Title/Grade/Due Date
